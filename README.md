# API-Connected Database App
Simple app that connects to an API, downloads some JSON, stores it in a database, and displays it back to the user.

e926 is a furry imageboard website based off danbooru. It archives and distributes furry artwork. Posts can be searched using a tag system and users can upvote, downvote, and comment on posts.

The API documentation is located at:

https://e926.net/help/api
